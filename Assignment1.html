Assignment 1 (C-Programming)

Download

Problem-Set:

     Design a flowchart/ Algorithm for a basic calculator that accepts two numbers and an operator (+, -, *, /) as input from the user and performs the corresponding operations, and displaying/print the result.


Objective:

     The objective of this question is to design a flowchart and algorithm for a basic calculator program. The program should be able to accept two numbers and an operator (+, -, *, /) as inputs from the user, perform the corresponding arithmetic operation, and display the result. This exercise aims to develop a clear understanding of fundamental programming concepts, such as input handling, conditional statements, and basic arithmetic operations.


Algorithm: 


                
                
1. Start
2. Ask the user if they want to "run" or "stop"
    - If "run", continue to step 3
    - If "stop", end the program
3. Clean the input buffer
4. Check the operator:
    - If "+" , go to step 6
    - If "-" , go to step 7
    - If "*" , go to step 8
    - If "/" , go to step 9
    - Otherwise, display "Invalid operator" and go to step 11
5. Input the first number (num1)
6. Input the second number (num2)                    
7. Result = Add(num1 + num2)
8. Result = Subtract(num1 - num2)
9. Result = Multiply(num1 * num2)
10. Result = Divide(num1 / num2)
    - if(num2 == 0) display("Error: Division by zero!")
11. Display(result)
12. end

Code: 


                
                
#include <stdio.h>
#include <string.h>
#include <stdlib.h>
#include <math.h>

                        
// Function Prototypes
void removespaces(char *);
void input(float *, float *);
int format(float, float);

                        
int main(void) {
    for (; ;) {
        // Infinite loop to continuously prompt user for input
        char check[10];

        printf("Enter your option - \n1. Run \n2. Stop\nEnter Choice: ");
        if (fgets(check, sizeof(check), stdin)) {
            // Read user input
            check[strcspn(check, "\n")] = '\0';
            // Remove newline character
            removespaces(check);
            // Remove any spaces in the input
            fflush(stdin);
            int swt = atoi(check); // Convert input to integer
            switch (swt) {
                case 1: {
                    // Case 1: Run the calculator
                    char str[10];
                    float num1, num2;
                    printf("Enter your choice by the option:\n + (for Addition)\n - (for subtraction)\n * (for multiplication)\n / (for division)\nEnter choice: ");
                    if (fgets(str, sizeof(str), stdin)) {
                        // Read operation choice
                        str[strcspn(str, "\n")] = '\0';
                        // Remove newline character
                        removespaces(str);
                        // Remove any spaces in the input
                        fflush(stdin);
                        char change = str[0];
                        // Get the first character of the input (operation)
                        switch (change) {
                            case '+': {
                                // Addition
                                input(&num1, &num2); // Get input numbers
                                int checker = format(num1, num2); // Determine format
                                if (checker == 1) {
                                    printf("Result: %d\n", (int)(num1 + num2)); // Print result as integer
                                } else if (checker == 0) {
                                    printf("Result: %.2f\n", num1 + num2); // Print result as float
                                } else {
                                    printf("Result: %.2f\n", (float)(num1 + num2)); // Print result as float
                                }
                                break;
                            
                                case '-': {
                                    // Subtraction
                                    input(&num1, &num2); // Get input numbers
                                    int checker = format(num1, num2); // Determine format
                                    if (checker == 1) {
                                        printf("Result: %d\n", (int)(num1 - num2)); // Print result as integer
                                    } else if (checker == 0) {
                                        printf("Result: %.2f\n", num1 - num2); // Print result as float
                                    } else {
                                        printf("Result: %.2f\n", (float)(num1 - num2)); // Print result as float
                                    }
                                    break;
                                
                                    case '*': {
                                        // Multiplication
                                        input(&num1, &num2); // Get input numbers
                                        int checker = format(num1, num2); // Determine format
                                        if (checker == 1) {
                                            printf("Result: %d\n", (int)(num1 * num2)); // Print result as integer
                                        } else if (checker == 0) {
                                            printf("Result: %.2f\n", num1 * num2); // Print result as float
                                        } else {
                                            printf("Result: %.2f\n", (float)(num1 * num2)); // Print result as float
                                        }
                                        break;
                                    
                                        case '/': {
                                            // Division
                                            input(&num1, &num2); // Get input numbers
                                            int checker = format(num1, num2); // Determine format
                                            if(!num2){
                                                printf("invalid\n");
                                            } else{
                                                if (num2 == 0) {
                                                    printf("Error: Division by zero\n"); // Check for division by zero
                                                } else {
                                                    float div = (float)(num1 / num2); // Perform division
                                                    double ini, fol;
                                                    fol = modf(div, &ini); // Separate integer and fractional parts
                                                    if (fol == 0){
                                                        printf("Result: %d\n", (int)(div)); // Print result as integer if no fractional part
                                                    } else {
                                                        printf("Result: %.2f\n", (float)(div)); // Print result as float
                                                    }
                                                }
                                            }
                                            break;
                                            default: {
                                                // Invalid operation input
                                                printf("Enter according to the options\n");
                                            } break;
                                            } else {
                                                printf("Error Reading Input!!\n");
                                            } break;
                                            case 2: {
                                                // Case 2: Stop the program
                                                exit(0);
                                            } break;
                                            default: {
                                                // Invalid menu input
                                                printf("Error! Enter 1 or 2 by option...\n");
                                            }
                                        }
                                    } else { 
                                            printf("Error Reading Input!!\n"); 
                                        }
                                        return 0; 
                                        }
                                        
                        
                                        // Function to remove spaces from a string
                                        void removespaces(char *str) {
                                            int count = 0;
                                            for (int i = 0; str[i]; i++) {
                                                if (str[i] != ' ') {
                                                    str[count++] = str[i];
                                                }
                                            }
                                            str[count] = '\0';
                                        }
                        
                                        // Function to take two float inputs from the user
                                        void input(float *a, float *b) {
                                            printf("You have to enter two numbers:\n number = ");
                                            if (scanf("%f", a) != 1) {
                                                printf("Error reading input\n");
                                                exit(1);
                                            }
                                            fflush(stdin);
                                            printf(" number = ");
                                            if (scanf("%f", b) != 1) {
                                                printf("Error reading input\n");
                                                exit(1);
                                            }
                                            fflush(stdin);
                                        }
                        
                                        // Function to determine if both numbers are integers, both are floats, or mixed
                                        int format(float a, float b) {
                                            double int_part1, int_part2, float_part1, float_part2;
                                            float_part1 = modf(a, &int_part1);
                                            float_part2 = modf(b, &int_part2);
                                            if (float_part1 == 0 && float_part2 == 0)
                                                return 1; // Both integers
                                            else if (float_part1 != 0 && float_part2 != 0)
                                                return 0; // Both floats
                                            else
                                                return -1; // Mixed
                                        }

Conclusion:

    This C program implements a basic calculator that can perform addition, subtraction, multiplication, and division operations. The program runs in an infinite loop, allowing the user to repeatedly choose to run the calculator or exit the program. Below are the key components and their functionalities:

1. Main Function:

  • Input Loop: Continuously prompts the user to choose between running the calculator or stopping the program.
    • Option Handling:
      • If the user chooses to run the calculator, it prompts for the operation (+, -, *, /) and two numbers.
      • Based on the chosen operation, it performs the corresponding arithmetic calculation and displays the result.
      • If the user chooses to stop the program, it exits the loop and ends the program.
      • If an invalid option is chosen, it prompts the user to enter a valid choice.
  • removespaces Function:
    • Purpose: Removes all spaces from a given string.
    • Usage: Cleans up user input to ensure no extra spaces interfere with processing.
  • input Function:
    • Purpose: Prompts the user to input two floating-point numbers.
    • Error Handling: If the user input is invalid (not a number), it displays an error message and exits the program.
  • format Function:
    • Purpose: Determines the format of the numbers (integers or floating-point numbers) based on their fractional parts.
    • Return Values:
      • Returns 1 if both numbers are integers.
      • Returns 0 if both numbers are floating-point numbers.
      • Returns -1 if one number is an integer and the other is a floating-point number.
  • Operation Handling:
    • For each operation (+, -, *, /), the program:
      • Calls the input function to get the two numbers from the user.
      • Calls the format function to determine the format of the numbers.
      • Performs the corresponding arithmetic operation.
      • Displays the result in the appropriate format (integer or floating-point).
    • Division Specific Handling:
      • Checks for division by zero and displays an error message if applicable.
  • Example Execution:
    • The user is prompted to choose an option (1 to run the calculator, 2 to stop).
    • If the user chooses to run the calculator:
      • The user is prompted to choose an arithmetic operation (+, -, *, /).
      • The user is prompted to enter two numbers.
      • The program performs the chosen operation and displays the result.
    • The loop continues until the user chooses to stop the program by entering 2.

This program effectively demonstrates handling user input, performing basic arithmetic operations, and managing various input and error scenarios in a console-based application.
